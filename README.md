# FINP - 智能问答知识库系统

基于 RAG (Retrieval-Augmented Generation) 技术的智能问答系统，支持多知识库管理和语义搜索。
 
## 🚀 系统架构

- **前端**: React + TypeScript + Antd + Vite
- **后端**: FastAPI + Python
- **向量数据库**: Qdrant
- **关系数据库**: MySQL
- **AI模型**: 通义千问 (DashScope)

## 📋 功能特性

- ✅ **多知识库管理**: 支持动态知识库选择和切换
- ✅ **智能问答**: 基于RAG技术的准确问答
- ✅ **语义搜索**: 支持混合搜索(语义+关键词)
- ✅ **文档处理**: 支持PDF、Word等多种格式文档解析
- ✅ **图表提取**: 自动提取文档中的表格和图片
- ✅ **聊天界面**: 友好的对话式交互界面
- ✅ **搜索历史**: 记录和管理搜索历史
